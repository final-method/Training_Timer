# Training_Timer
A simple timer to track training hours... If your company needs tracking :)
This is just a simple training tracker. it doesn't do much more then that. There is a place to start up the tacker, start, pause, and stop and a place for notes
When you hit stop it will generate an excel file showing the training that you did for that date based on your nodes with a time stamp. 
Great for tracking off the clock if your company wants to know the time you have placed into personal devleopment :)
